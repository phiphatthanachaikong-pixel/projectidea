# projectidea
ชื่อโปรเจกต์: ProjectIdea

##คำอธิบาย:
ProjectIdea เป็นโปรแกรมสำหรับบันทึกไอเดียโครงงาน ช่วยให้ผู้ใช้สามารถเพิ่ม แก้ไข และดูรายละเอียดไอเดียโครงงานได้อย่างเป็นระบบ เหมาะสำหรับใช้เก็บรวบรวมแนวคิดก่อนนำไปพัฒนาเป็นโครงงานจริง

โครงงานนี้เหมาะสำหรับ:
1.นักเรียน / นักศึกษา ที่ต้องทำโครงงาน
2.ผู้ที่กำลังหาไอเดียทำโปรเจกต์
3.ผู้พัฒนาโปรแกรมที่ต้องการเก็บแนวคิดของตนเอง
4.อาจารย์ที่ต้องการใช้เป็นตัวอย่างการเรียนการสอน C#

เทคโนโลยีที่ใช้ในโปรเจกต์:
ภาษาโปรแกรม: C#
รูปแบบแอปพลิเคชัน: Windows Forms
Framework: .NET
เครื่องมือพัฒนา: Visual Studio

วิธีดาวน์โหลด (Clone) โค้ดจาก GitHub สามารถดาวน์โหลดโค้ดได้จากแพลตฟอร์ม GitHub โดยทำตามขั้นตอนดังนี้:

วิธีที่ 1: ดาวน์โหลดเป็นไฟล์ ZIP
เข้าไปที่หน้า Repository
กดปุ่ม Code
เลือก Download ZIP
แตกไฟล์ลงในเครื่องคอมพิวเตอร์

วิธีที่ 2: ใช้คำสั่ง Git Clone
คัดลอก URL ของ Repository
เปิด Git Bash หรือ Command Prompt
พิมพ์คำสั่ง git clone https://github.com/phiphatthanachaikong-pixel/projectidea.git
วิธีการเปิดโปรเจกต์ และวิธีการรัน

วิธีเปิดโปรแกรม Visual Studio:
เลือกเมนู Open a project or solution
ไปยังโฟลเดอร์ที่ดาวน์โหลดมา
เลือกไฟล์ .sln
กดปุ่ม Start หรือกด F5 เพื่อรันโปรแกรม
เมื่อรันแล้ว หน้าจอโปรแกรมจะปรากฏขึ้นพร้อมใช้งาน

คำอธิบายหน้าจอโปรแกรม:
IdeaID
ใช้สำหรับกรอกรหัสไอเดีย (เลขประจำรายการ)
IdeaTitle
ใช้สำหรับกรอกชื่อหัวข้อไอเดียโครงงาน
category
ใช้สำหรับเพิ่มและกำหนดหมวดหมู่ของไอเดีย
status
ใช้สำหรับเพิ่มและกำหนดสถานะของไอเดีย
ปุ่ม Add
ใช้สำหรับบันทึกข้อมูลไอเดีย

ตัวอย่างขั้นตอนการเพิ่มไอเดีย:
กรอก IdeaID เช่น 001
กรอก IdeaTitle เช่น ระบบจัดการร้านกาแฟ
กรอกข้อมูลหรือเลือก category จาก combobox
กรอกข้อมูลหรือเลือก status จาก combobox
กดปุ่ม Add หรือ Save
หากผู้ใช้ไม่กรอกข้อมูลในช่อง Idea_id หรือ ide_titleแล้วกดปุ่ม ADD จะขึ้นแจ้งเตือน
หากผู้ใช้กรอกข้อมุลครบแล้วกดปุ่มADDโปรแกรมจะแสดง MessageBox เสรุปข้อมูล

คำอธิบายการใช้ปุ่ม Clear และ Exit:
ปุ่ม Clear
ใช้ล้างข้อมูลทั้งหมดใน TextBox
เหมาะสำหรับกรอกข้อมูลใหม่
ไม่กระทบข้อมูลที่บันทึกไปแล้ว
ปุ่ม Exit
ใช้สำหรับออกจากโปรแกรม
เมื่อกดแล้ว โปรแกรมจะปิดทันที
